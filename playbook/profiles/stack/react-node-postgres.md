## React + Node.js + PostgreSQL Stack Profile

## Zakres
- frontend: React + TypeScript
- backend: Node.js (Express/Nest)
- baza danych: PostgreSQL

## Setup platform-specific
- skonfiguruj package manager (npm/pnpm)
- skonfiguruj build frontend i backend przez CLI
- skonfiguruj test runner (Vitest/Jest)
- skonfiguruj lint/format (ESLint + Prettier)
- skonfiguruj migracje DB

## Script expectations
- `Scripts/build.sh` buduje frontend i backend
- `Scripts/test.sh` uruchamia testy unit/integration
- `Scripts/lint.sh` uruchamia lint i format-check
